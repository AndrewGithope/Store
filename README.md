# 🛒 Shop for All — Angular Online Store

A modern, responsive e-commerce web application built with **Angular 22** featuring an interactive **AI Shopping Assistant** powered by the **Google Gemini API**.

---

## ✨ Features

* **📦 Product Catalog:** Browse products across multiple categories with dynamically loaded inventory and clean UI layouts.
* **🤖 AI Shopping Assistant:** Integrated Gemini-powered chat widget that assists users with product discovery and custom recommendations based on the live catalog.
* **🛒 Interactive Shopping Cart:** Add, remove, and manage item quantities seamlessly using reactive state management.
* **⚡ Reactive State & Signals:** High-performance UI reactivity using Angular Signals.
* **📱 Responsive Design:** Modern and mobile-friendly UI layout.

---

## 🛠️ Tech Stack

* **Frontend Framework:** Angular 22 (Standalone Components, Signals)
* **AI Integration:** Google Generative AI SDK (`@google/generative-ai`) — Gemini 1.5 Flash
* **State & Asynchronous Logic:** RxJS & Angular Signals
* **Forms & Styling:** Angular FormsModule, CSS3 / Flexbox / Grid
